# AI Programming Lab Mentor

## Project Overview
AI Programming Lab Mentor is an AI-powered virtual assistant that helps students learn programming through code explanations, debugging assistance, quizzes, and personalized guidance.

## Objectives
- Help students understand programming concepts.
- Explain code in simple language.
- Assist in debugging programs.
- Generate programming exercises and quizzes.
- Support self-paced learning.

## Technologies Used
- Python
- IBM watsonx.ai
- GitHub
- Generative AI
